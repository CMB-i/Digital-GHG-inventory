Getting Started

Formulas are always attached to calculated fields and cannot be configured in isolation. Please open the formula builder from the sheet builder layout instead.

1. Connected to Sheets

Formulas are bound directly to calculated fields in your sheets (e.g. converting MWh to tCO₂e or summing monthly values).

2. Launch from Sheet Builder

To configure or edit a formula, go to the sheet layout designer, add/select a calculated field, and click Open Formula Builder.

3. Immutable Versions

Once published, formula versions are immutable to preserve audit trails. Any sheet referencing them stays locked to that version.

4. Versioning Iterations

To update a calculation, save draft changes and publish a new version. You can then link the new version in your sheet builder field inspector.

01. Drag or Click Operands

Locate the numeric variables under the Fields tab or Value Set constants under the Value Sets tab. You can click them to insert at the cursor, or drag and drop them directly onto the LCD screen.

02. Drop Math Keys

Drag calculator math keys (`+`, `-`, `*`, `/`) or click them to build arithmetic expressions. Use parentheses to control execution hierarchy.

03. Sheet Aggregation

Use the SUM MONTHS function to sum a monthly input across the financial year (for below-table aggregates):

Examples: • SUM_MONTHS(from_grid) • SUM_MONTHS(diesel) * diesel_ef

💡 You can drag and drop a Field chip directly onto the Σ SUM button to wrap it instantly.

05. Test Calculations

Expand the bottom drawer. Type values for each detected variable to see live results immediately. Click Validate Syntax to run formal backend parser checks.
